0.2 Create Dashboard

Adafruit provide a front end Dashboard to display the data online in real time.
It is a cloud server that can be used to connect to devices through wifi and also to control these devices through it's dashboard. A Dashboard will act as a medium to display the values of the feeds/sensor values that you are sending

Here we will select Dashboard from the top bar and that will display all of the information regarding dashboards that you have created. Now, you can click on the New Dashboard Button anf fill in the Dashboard name as per your liking and the then click on create once done.

The list should now display the generated dashboard. If you click on it you can see the live version of the dashboard at the moment it should be blank which we will populate in later steps.

0.3 Adding Feeds

Now that the dashboard is created , let's start ,creating feeds to populate it. Feeds are basically the name of the sensor value that we are sending from the IAQ board.
Let's Start with Temperature feed. To do that find feeds from the top bar that will take you to the adafruit feeds tab. Now click on create New Feed button, this will show you a New Feed's menue.

0.4 Link Feeds To The Dashboard

Once the feed and the dashboard is create now it's time to create a circular meter for Trmperature and Humidity that links to the created respective feeds. Navigate to the dashboard tab and select the dashboard created in previous steps. Click the settings button and then select Create new block. This will open up a menue with all of the options of that you can add to the dashboard, select gauge. Finally, you can link the feeds by selecting the name of the feed lets choose Temperature first. Add the following information in the gauge settings:

  1. Block Title: Temperature
  2. Gauge Min Value: 0 (This indicates the minimum value of the Temperature)
  3. Gauge Max Value: 70 (This indicates the maximum value of the Temperature)
  4. Gauge Label: C (Indicating Celcius)
  5. High Warning Value: 30 (Once this value is crossed, the dial will change colour and will even send a alert once set)

Then click Create block This can be seen as below.

Now let's follow the same steps as above and choose the feed as Humidity. Add the following information in the gauge settings:

  1. Block Title: Humidity
  2. Gauge Min Value: 0 (This indicates the minimum value of the Humidity)
  3. Gauge Max Value: 100 (This indicates the maximum value of the Humidity)
  4. Gauge Label: % (Indicating Percentage)
  5. High Warning Value: 45 (Once this value is crossed, the dial will change colour and will even send a alert once set)

Then click Create block The outcome will be like below.

Finally, create two other buttons for led and buzzer.

0.5 Edit The Dashboard (OPTIONAL)

This is just an optional step if you want to edit the way your dashboard look. To do this, click on the same settings button and now select Edit Layout which will display setting symbol on top left corner of the block now simply drag the block to align them. You can even adjust the size of the block by placing the mouse on bottom left corner of the block and dragging the corner. Once Satisfied click on Save Layout.
